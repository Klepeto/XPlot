Deedle Helpers

Install-Package XPlot.GoogleCharts.Deedle

This package allows you to visualize Deedle series and data frames.

Bootstrapping Script

The package comes with a bootstrapping script that references the necessary components and instructs FSI to use the IE11 Standards mode for WPF browser controls thus preventing Google Charts from falling back to VML. If you're using NuGet then load the XPlot.GoogleCharts.fsx script. If you're managing packages with Paket then load the XPlot.GoogleCharts.Paket.fsx script.
